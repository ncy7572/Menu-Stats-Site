# Privacy Policy for Menu Stats

Last updated: March 20, 2026

Menu Stats is a macOS menu bar utility that displays system metrics including CPU, GPU, memory, swap, and network usage.

## Data Collection

Menu Stats does not collect, store, transmit, or share personal data or usage information.

## What the App Accesses

The app reads the following information locally on your device only to display system activity:

- CPU usage
- GPU usage on supported Apple Silicon Macs
- Memory usage
- Swap usage
- Network throughput

This information never leaves your device.

## Analytics and Tracking

Menu Stats does not include analytics, advertising SDKs, crash reporting services, or third-party tracking.

## External Services

Menu Stats does not use external servers, cloud sync, or third-party APIs.
